Istruzioni e operatori

La riga di codice:

int errore = NO_ERRORI;

è un’istruzione. Le istruzioni sono composte da una o più espressioni, composte a loro volta da operatori e operandi. Le espressioni possono essere o semplici o composte e producono sempre un valore:

a = 2 
a + b 
d = a - b + c 
a < d

La differenza fra istruzioni ed espressioni è che le istruzioni sono sempre terminate dal carattere ;:

a = 3; 
b = a * 3; 
c = sqrt(b);
